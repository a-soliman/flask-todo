# Todo RESTful API
## TECHNOLOGIES: 
* Python3 
* Flask
* Flask-RESTful
* SQLite3,
* QLAlchemy
* Virtualenv

## INSTALLITION:
1. Install Python3
    [Python website](https://www.python.org/)
1. Install Flask
    [Flask website](http://flask.pocoo.org/)
1. Install Virtualenv
    ```bash
    sudo pip3 install virtualenv
    ```
1. Clone this Repo
    ```bash
    git clone <repo url>
    ```
1. activate Virtualenv
    ```bash
    source venv/bin/activate
    ```
1. Install the required packages if needed
    ```bash
    pip install ...

    ```
1. Run the App
    ```bash
    python app.py
    ```
